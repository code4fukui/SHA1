# SHA1

SHA1はファイルのハッシュ値を計算するツールです。

## デモ
ターミナルから以下のように使えます:

```
$ deno run --allow-read cli.js file.txt
2aae6c35c94fcfb415dbe95f408b9ce91ee846ed
```

## 機能
- ファイルのSHA1ハッシュ値を計算する
- 文字列のSHA1ハッシュ値を計算する

## 必要環境
- Deno 1.8以上

## 使い方
リポジトリをクローンし、以下のように使ってください:

```
$ git clone https://github.com/taisukef/SHA1.git
$ cd SHA1
$ deno run --allow-read cli.js file.txt
```
